Ediz

Ik heb de website geprogrammeerd. Dit heb ik gedaan met de talen HTML, CSS en JavaScript. Ik had al een beetje ervaring met programmeren, maar helemaal geen ervaring met webdevelopment. Ik ben begonnen met HTML door YouTube-tutorials te volgen. Daarna ben ik doorgegaan naar CSS, wat ik het moeilijkst vond. JavaScript was iets makkelijker voor mij, omdat ik al ervaring had met talen zoals Python en C#, die een beetje lijken op JavaScript. Jammer genoeg was voor deze website niet heel veel JavaScript nodig, maar juist heel veel CSS en HTML. Het programmeren van deze website hoort bij het vak Informatica, omdat je daar ook programmeren leert. Ik vond het vooral belangrijk om te leren hoe ik een website kan programmeren zonder de hulp van ChatGPT en het internet. Het internet heb ik nog steeds nodig voor sommige dingen, maar ik denk dat ik de basis van webdevelopment best goed heb kunnen leren.
