Troy Bussink

Ik ben 17 jaar oud, en woon in Eibergen In dit profielwerkstuk heb ik de informatie verzameld, de vragen gemaakt, de concepten voor het User interface gemaakt, en zelf zonder het te weten meegeholpen met programmeren. Het belangrijkste wat ik hiervoor gedaan heb is het onderzoeken van Havo examenstof door het “Geschiedenis Werkplaats” boek te bestuderen. Ons profielwerkstuk is voor mij relevant bij het vak Geschiedenis, Ik hoopte met dit profielwerkstuk een goed beeld te krijgen van de examenstof en een makkelijke plek te creëren om deze stof te leren. Je zou denken dat ik iets opgepikt heb nu ik een Word bestand met meer dan 11.000 woorden heb, wat de samenvatting en de vragen bevat. Dit was zeker een interessant project om aan te werken.

Ediz

ik heb het website geprogrameerd, dit heb ik met de talen HTML, CSS en JavaScript gedaan. Ik had al een beetje ervaring met programeren, maar helemaal geen ervaring met web-developing. Ik ben begonnen met HTML door youtube tutorials te volgen, daarna ben ik doorgegaan naar CSS wat ik het moeilijkste vind. JavaScript was iets makkelijker voor mij sinds dat ik al ervaring had met talen zoals Python en C# die een beetje lijken op JavaScript. Jammergenoeg was voor deze website niet heel veel JavaScript nodig maar juist heel veel CSS en HTML. Dit website programeren hoort bij het vak Informartica, sinds dat je daar ook programeren leert. Ik vond het vooral belangerijk om te leren hoe ik een website kan programeren zonder de hulp van chatGPT en de Internet. De internet heb ik nogsteeds nodig voor sommige dingen maar ik denk dat ik de basis van web-developing best goed heb kunnen leren.
